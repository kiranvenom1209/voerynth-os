# Wake Word Model

## Required File

Place the Porcupine wake word model file here:

**Filename:** `Hey-Ammu_en_android_v4_0_0.ppn`

**Location:** `android/app/src/main/assets/Hey-Ammu_en_android_v4_0_0.ppn`

## Instructions

1. Copy the provided `Hey-Ammu_en_android_v4_0_0.ppn` file to this directory
2. The file should be at the same level as this README
3. The WakeWordPlugin will load it from the assets at runtime

## Verification

After placing the file, this directory should contain:
- `Hey-Ammu_en_android_v4_0_0.ppn` (the wake word model)
- `README_WAKE_WORD.md` (this file)
- Other Capacitor assets (capacitor.config.json, etc.)
