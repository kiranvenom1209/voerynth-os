### 1. **Code License: Apache 2.0**
The source code is licensed under Apache License 2.0, which allows:

✅ **Commercial use** - Others can use the code in commercial products  
✅ **Modification** - Others can modify and adapt the code  
✅ **Distribution** - Others can distribute the code  
✅ **Patent use** - Grants patent rights  
✅ **Private use** - Can be used privately  

**Requirements for users:**
- Must include the original copyright notice
- Must include the Apache 2.0 license text
- Must state significant changes made to the code
- Must include the NOTICE file if distributing

### 2. **Trademark Protection: Exclusive Rights**
The brand name and identity are **exclusively owned** by you:

🔒 **"Vœrynth OS"** - Protected trademark  
🔒 **"Vœrynth"** - Protected trademark

🔒 **Design elements** - Protected visual identity  

**What this means:**

- But they **CANNOT** use the **"Vœrynth" name or branding**
